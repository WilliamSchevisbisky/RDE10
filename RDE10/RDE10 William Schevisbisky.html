Cascading Style Sheets (CSS)

CSS - Definição

É um mecanismo para adicionar estilização à um site na Web, permitindo a alteração das cores, alinhamento e fontes. A esmagadora maioria dos sites modernos utilizam esse recurso para melhorar a experiência do usuário fazendo um site mais bonito e agradável ao olhos.

Componentes

  • Seletores - Seleciona elementos HTML com base no nome do elemento ao qual o estilo será aplicado.
  • Propiedades - O aspecto especifico do elemento que você quer usar como estilo.
  • Valores - A configuração aplicada ao atributo.

Formas de utilização

CSS - Inline

Consiste em colocar a estilização diretamento no componente Pode ser usado para alterações pontuais e simples, no geral apesar de funcionar, é algo que devemos evitar usar fora dessa situação pois atrapalha bastante a manutenção em um site, ja que para alterar o estilo, seria necessario ir componente por componente mudando seu atributo 'Style'

CSS - Incorporado

Também chamado de Internal, é utilizar a estilização dentro da tag: É dessa maneira que este site esta sendo implementado, é usado quando a estilização sera atrelada apenas a um arquivo, e de maneira geral, em projetos menos complexos. A refatoração para o modo 'Externo' que é o mais aconselhado, pode ser feita de maneira bastante simples copiando o conteudo da tag para um arquivo externo .css

CSS - Externo

Consiste em colocar a estilização em um arquivo a separado do HTML Você pode colocar toda a sua estilização em um arquivo .css separado, depois de fazer isso basta usar a tag: com os parametros rel="stylesheet" e href="caminho/arquivo.css"
